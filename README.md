# LeadOps Intelligence

A modern lead intelligence platform for discovering, scoring, and managing business prospects.

## Features

- **Dashboard** — Real-time pipeline analytics with charts and KPIs
- **Lead Search** — Full-text search with advanced filtering by industry, location, size, revenue, and score
- **Lead Scoring** — Visual quality scores (hot / warm / cold) for every prospect
- **Lead Details** — Comprehensive company and contact profiles
- **Saved Leads** — Bookmark and revisit your top opportunities
- **CSV Export** — One-click download of your lead database

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Recharts (data visualization)
- React Router (SPA routing)
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```
